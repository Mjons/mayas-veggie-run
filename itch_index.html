Maya's Veggie Run

This game requires Python and Pygame to run.
Since it uses native Pygame (not web-compatible), it needs to be run locally on your computer.

How to Play Locally

  1. Download the game files from the repository
  2. Install Python 3.7+ and Pygame: 
    pip install pygame
  3. Run the game: 
    python Maya.py

Game Controls

SPACE: Jump

Goal: Collect veggies to maintain energy

Special: Find the puppy for full health!
